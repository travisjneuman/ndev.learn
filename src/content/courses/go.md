## Coming Soon

The language behind Docker, Kubernetes, and most of the cloud-native ecosystem. Go's simplicity, fast compilation, and first-class concurrency make it ideal for building services, CLIs, and infrastructure tooling.

This course covers Go from fundamentals through goroutines and channels to building production cloud services.

## Top Resources to Get Started

### Official Documentation
- [A Tour of Go](https://go.dev/tour/) — Official interactive Go tutorial
- [Effective Go](https://go.dev/doc/effective_go) — Idiomatic Go patterns and style
- [Go by Example](https://gobyexample.com/) — Annotated example programs

### Interactive Courses
- [Exercism Go Track](https://exercism.org/tracks/go) — Mentored Go coding exercises
- [Go Track on Boot.dev](https://www.boot.dev/tracks/backend) — Backend development course in Go
- [Gophercises](https://gophercises.com/) — Coding exercises for Go developers
- [Learn Go with Tests](https://quii.gitbook.io/learn-go-with-tests) — TDD approach to learning Go

### Video & Guided Learning
- [Fireship Go in 100 Seconds](https://www.youtube.com/watch?v=446E-r0rXHI) — Quick overview
- [Anthony GG's Go Tutorials](https://www.youtube.com/c/AnthonyGG) — Practical Go projects
- [TechWorld with Nana — Go](https://www.youtube.com/watch?v=yyUHQIec83I) — Go for beginners

### Books
- [The Go Programming Language](https://www.gopl.io/) — Donovan & Kernighan's comprehensive guide
- [Let's Go](https://lets-go.alexedwards.net/) — Build professional web apps in Go
- [Concurrency in Go](https://www.oreilly.com/library/view/concurrency-in-go/9781491941294/) — Master goroutines and channels
