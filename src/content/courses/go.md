## Coming Soon

The language behind Docker, Kubernetes, and most of the cloud-native ecosystem. Go's simplicity, fast compilation, and first-class concurrency make it ideal for building services, CLIs, and infrastructure tooling.

This course covers Go from fundamentals through goroutines and channels to building production cloud services.

## Top Resources to Get Started

### Official & Core Documentation
- [A Tour of Go](https://go.dev/tour/) — Official interactive tutorial teaching Go fundamentals in the browser
- [Effective Go](https://go.dev/doc/effective_go) — Idiomatic Go patterns, style, and conventions from the Go team
- [Go by Example](https://gobyexample.com/) — Annotated example programs covering the entire language
- [Go Standard Library Documentation](https://pkg.go.dev/std) — Complete API reference for Go's rich standard library
- [Go Specification](https://go.dev/ref/spec) — The formal language specification
- [Go Blog](https://go.dev/blog/) — Official blog with deep dives on language features, design decisions, and best practices

### Interactive Courses & Platforms
- [Learn Go with Tests](https://quii.gitbook.io/learn-go-with-tests) — Free TDD-based approach to learning Go, widely recommended by the community
- [Exercism Go Track](https://exercism.org/tracks/go) — Mentored coding exercises with community feedback
- [Boot.dev — Go Track](https://www.boot.dev/tracks/backend) — Backend development course built around Go
- [Gophercises](https://gophercises.com/) — Free coding exercises designed specifically for Go developers
- [Go Playground](https://go.dev/play/) — Write and run Go code directly in the browser
- [Practical Go Lessons](https://www.practical-go-lessons.com/) — Free, comprehensive Go introduction for all levels
- [Codewars Go](https://www.codewars.com/?language=go) — Progressive coding challenges in Go
- [Getting Started with Go (Coursera)](https://www.coursera.org/learn/golang-getting-started) — University of California, Irvine course, free to audit

### Books & Long-Form Reading
- [The Go Programming Language](https://www.gopl.io/) — Donovan and Kernighan's comprehensive guide, the definitive Go book
- [Let's Go](https://lets-go.alexedwards.net/) — Alex Edwards' practical guide to building professional web applications in Go
- [Let's Go Further](https://lets-go-further.alexedwards.net/) — Sequel covering advanced API patterns, JSON handling, and deployment
- [Learning Go](https://www.oreilly.com/library/view/learning-go-2nd/9781098139285/) — Jon Bodner's modern Go guide, updated for Go 1.21+ features
- [Concurrency in Go](https://www.oreilly.com/library/view/concurrency-in-go/9781491941294/) — Master goroutines, channels, and concurrent patterns
- [Go in Action](https://www.manning.com/books/go-in-action) — William Kennedy's practical introduction covering real-world Go development
- [Boring Go!](https://www.manning.com/books/boring-go) — Modern Go as it's actually written in production, the top-recommended Go book for 2026
