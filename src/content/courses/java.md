## Coming Soon

The language that runs half the world's enterprise software. Backend (Spring Boot), Android development, distributed systems, and cloud infrastructure — Java's ecosystem is massive and battle-tested.

This course covers modern Java from fundamentals through Spring Boot microservices and Android app development.

## Top Resources to Get Started

### Official & Core Documentation
- [Oracle Java Tutorials](https://docs.oracle.com/javase/tutorial/) — Comprehensive official tutorials covering language basics through advanced topics
- [Java SE API Documentation](https://docs.oracle.com/en/java/javase/21/docs/api/) — Complete API reference for the Java standard library
- [Spring Boot Guides](https://spring.io/guides) — Hands-on quickstart guides for building Spring Boot applications
- [Spring Boot Reference Documentation](https://docs.spring.io/spring-boot/reference/) — Official reference for the most popular Java framework
- [Android Developer Guides](https://developer.android.com/guide) — Official Android development documentation and best practices
- [OpenJDK](https://openjdk.org/) — The open-source reference implementation of the Java platform

### Interactive Courses & Platforms
- [MOOC.fi Java Programming](https://java-programming.mooc.fi/) — University of Helsinki's free, comprehensive Java course (widely considered the best free option)
- [JetBrains Academy — Java](https://hyperskill.org/tracks/1) — Project-based Java learning with real-world applications
- [Exercism Java Track](https://exercism.org/tracks/java) — Mentored coding exercises with community feedback
- [Codecademy Learn Java](https://www.codecademy.com/learn/learn-java) — Interactive beginner course with in-browser coding
- [CodingBat Java](https://codingbat.com/java) — Free practice problems for Java fundamentals and string/array logic
- [Codewars Java](https://www.codewars.com/?language=java) — Thousands of coding challenges from beginner to expert level
- [Java Design Patterns](https://java-design-patterns.com/) — Open-source catalog of design patterns implemented in Java
- [Baeldung](https://www.baeldung.com/) — Thousands of free tutorials on Java, Spring, and related technologies

### Books & Long-Form Reading
- [Effective Java](https://www.oreilly.com/library/view/effective-java-3rd/9780134686097/) — Joshua Bloch's essential Java best practices, the must-read for every Java developer
- [Head First Java](https://www.oreilly.com/library/view/head-first-java/9781492091646/) — Visual, hands-on introduction to Java programming
- [Modern Java in Action](https://www.manning.com/books/modern-java-in-action) — Lambdas, streams, functional patterns, and reactive programming
- [Java Concurrency in Practice](https://jcip.net/) — Brian Goetz's definitive guide to writing thread-safe Java code
- [Spring in Action](https://www.manning.com/books/spring-in-action-sixth-edition) — Comprehensive guide to the Spring ecosystem
- [Clean Code](https://www.oreilly.com/library/view/clean-code-a/9780136083238/) — Robert C. Martin's principles of writing readable, maintainable code (Java-focused examples)
