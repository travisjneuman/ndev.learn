## Coming Soon

The language that runs half the world's enterprise software. Backend (Spring Boot), Android development, distributed systems, and cloud infrastructure — Java's ecosystem is massive and battle-tested.

This course covers modern Java from fundamentals through Spring Boot microservices and Android app development.

## Top Resources to Get Started

### Official Documentation
- [Oracle Java Tutorials](https://docs.oracle.com/javase/tutorial/) — Comprehensive official tutorials
- [Spring Boot Guides](https://spring.io/guides) — Hands-on Spring Boot quickstarts
- [Android Developer Guides](https://developer.android.com/guide) — Official Android documentation

### Interactive Courses
- [JetBrains Academy — Java](https://hyperskill.org/tracks/1) — Project-based Java learning
- [Exercism Java Track](https://exercism.org/tracks/java) — Mentored coding exercises
- [MOOC.fi Java Programming](https://java-programming.mooc.fi/) — University of Helsinki's free course
- [Codecademy Learn Java](https://www.codecademy.com/learn/learn-java) — Interactive beginner course

### Video & Guided Learning
- [Java Brains](https://www.youtube.com/c/JavaBrainsChannel) — Spring framework and Java tutorials
- [Amigoscode Java](https://www.youtube.com/c/amigoscode) — Modern Java and Spring Boot content
- [Coding with John](https://www.youtube.com/c/CodingwithJohn) — Clear Java concept explanations

### Books
- [Effective Java](https://www.oreilly.com/library/view/effective-java-3rd/9780134686097/) — Joshua Bloch's essential Java best practices
- [Head First Java](https://www.oreilly.com/library/view/head-first-java/9781492091646/) — Visual introduction to Java programming
- [Modern Java in Action](https://www.manning.com/books/modern-java-in-action) — Lambdas, streams, and functional patterns
