## Coming Soon

The foundation of modern infrastructure. Linux runs servers, containers, cloud instances, and developer machines worldwide. Master the command line, write shell scripts, and administer Unix systems with confidence.

This course builds from basic navigation through shell scripting to system administration and server management.

## Top Resources to Get Started

### Official Documentation
- [Linux man pages](https://man7.org/linux/man-pages/) — Comprehensive command reference
- [GNU Bash Manual](https://www.gnu.org/software/bash/manual/) — Official Bash shell documentation
- [Ubuntu Server Guide](https://ubuntu.com/server/docs) — Server administration docs

### Interactive Courses
- [Linux Journey](https://linuxjourney.com/) — Free, step-by-step Linux fundamentals
- [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) — Learn Linux through a fun wargame
- [Exercism Bash Track](https://exercism.org/tracks/bash) — Mentored Bash scripting exercises
- [Linux Survival](https://linuxsurvival.com/) — Interactive command-line tutorial
- [Terminus](https://web.mit.edu/mprat/Public/web/Terminus/Web/main.html) — Browser-based CLI adventure game

### Video & Guided Learning
- [The Missing Semester (MIT)](https://missing.csail.mit.edu/) — Command line, shell, and dev tools course
- [Linux for Hackers (NetworkChuck)](https://www.youtube.com/playlist?list=PLIhvC56v63IJIujb5cyE13oLuyORZpdkL) — Practical Linux series
- [Learn Linux TV](https://www.youtube.com/c/LearnLinuxtv) — Linux tutorials and server administration

### Books
- [The Linux Command Line](https://linuxcommand.org/tlcl.php) — Free online book, excellent for beginners
- [How Linux Works](https://nostarch.com/howlinuxworks3) — Understanding the internals
- [Linux Bible](https://www.wiley.com/en-us/Linux+Bible-p-9781119578888) — Comprehensive Linux reference
