## Coming Soon

The foundation of modern infrastructure. Linux runs servers, containers, cloud instances, and developer machines worldwide. Master the command line, write shell scripts, and administer Unix systems with confidence.

This course builds from basic navigation through shell scripting to system administration and server management.

## Top Resources to Get Started

### Official & Core Documentation
- [Linux man pages](https://man7.org/linux/man-pages/) — Comprehensive command reference for every Linux utility
- [GNU Bash Manual](https://www.gnu.org/software/bash/manual/) — Official Bash shell documentation and scripting reference
- [Ubuntu Server Guide](https://ubuntu.com/server/docs) — Server administration, networking, and security docs
- [Arch Wiki](https://wiki.archlinux.org/) — Community-maintained Linux knowledge base, useful for all distros
- [TLDP Advanced Bash-Scripting Guide](https://tldp.org/LDP/abs/html/) — In-depth Bash scripting reference with examples
- [Filesystem Hierarchy Standard](https://refspecs.linuxfoundation.org/FHS_3.0/fhs-3.0.html) — Understand the Linux directory structure

### Interactive Courses & Platforms
- [Linux Journey](https://linuxjourney.com/) — Free, step-by-step Linux fundamentals from grasshopper to networking
- [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) — Learn Linux through a wargame, progressively harder challenges
- [Linux Survival](https://linuxsurvival.com/) — Interactive command-line tutorial with a built-in console simulator
- [Exercism Bash Track](https://exercism.org/tracks/bash) — Mentored Bash scripting exercises with community code review
- [Terminus](https://web.mit.edu/mprat/Public/web/Terminus/Web/main.html) — Browser-based CLI adventure game from MIT
- [Introduction to Linux (LFS101)](https://training.linuxfoundation.org/training/introduction-to-linux/) — Free Linux Foundation course with certificate
- [Codecademy — Learn the Command Line](https://www.codecademy.com/learn/learn-the-command-line) — Interactive CLI basics in the browser
- [Cisco NDG Linux Unhatched](https://www.netacad.com/courses/os-it/ndg-linux-unhatched) — Free entry-level Linux course with skills badge

### Video Courses & YouTube Channels
- [The Missing Semester (MIT)](https://missing.csail.mit.edu/) — Command line, shell scripting, vim, and dev tools course
- [Learn Linux TV](https://www.youtube.com/c/LearnLinuxtv) — Comprehensive Linux tutorials and server administration guides
- [Linux for Hackers (NetworkChuck)](https://www.youtube.com/playlist?list=PLIhvC56v63IJIujb5cyE13oLuyORZpdkL) — Practical, energetic Linux series
- [tutoriaLinux](https://www.youtube.com/c/tutoriaLinux) — Linux administration tutorials and career guidance
- [DistroTube](https://www.youtube.com/c/DistroTube) — Linux distributions, window managers, and command-line tools
- [Chris Titus Tech](https://www.youtube.com/c/ChrisTitusTech) — Linux desktop and server tutorials with practical tips
- [The Linux Experiment](https://www.youtube.com/c/TheLinuxExperiment) — Accessible Linux desktop news, tutorials, and app reviews

### Books & Long-Form Reading
- [The Linux Command Line](https://linuxcommand.org/tlcl.php) — Free online book by William Shotts, excellent for beginners
- [How Linux Works (3rd Ed.)](https://nostarch.com/howlinuxworks3) — Deep understanding of Linux internals by Brian Ward
- [Linux Bible (10th Ed.)](https://www.wiley.com/en-us/Linux+Bible-p-9781119578888) — Comprehensive reference from basics to enterprise administration
- [UNIX and Linux System Administration Handbook](https://www.admin.com/) — Industry-standard sysadmin reference by Evi Nemeth et al.
- [Introduction to Bash Scripting](https://github.com/bobbyiliev/introduction-to-bash-scripting) — Free open-source ebook for Bash beginners
- [Mastering Linux Shell Scripting](https://www.packtpub.com/product/mastering-linux-shell-scripting-second-edition/9781788990554) — From basics to advanced automation scripts

- [LinuxCommunity.io](https://linuxcommunity.io/) — Forum for enthusiasts and sysadmins to collaborate
- [Linux.org Forums](https://www.linux.org/forums/) — Community forums covering all major distributions
