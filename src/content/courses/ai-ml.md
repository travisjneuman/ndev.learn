## Coming Soon

Build intelligent systems that learn from data. From classical machine learning algorithms through deep neural networks to modern LLM applications — understand how AI works and build with it.

This course covers ML fundamentals, PyTorch, transformer architectures, RAG pipelines, and deploying AI-powered applications in production.

## Top Resources to Get Started

### Official Documentation
- [PyTorch Tutorials](https://pytorch.org/tutorials/) — Official deep learning tutorials
- [Hugging Face Documentation](https://huggingface.co/docs) — Transformers, datasets, and model hub
- [OpenAI API Documentation](https://platform.openai.com/docs/) — GPT and DALL-E API reference
- [LangChain Documentation](https://python.langchain.com/docs/) — LLM application framework

### Interactive Courses
- [fast.ai](https://www.fast.ai/) — Practical deep learning, free and project-first
- [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) — Free ML fundamentals
- [Kaggle Competitions](https://www.kaggle.com/competitions) — Real ML challenges with datasets
- [Hugging Face Course](https://huggingface.co/learn/nlp-course) — NLP with Transformers

### Video & Guided Learning
- [3Blue1Brown — Neural Networks](https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi) — Visual neural network explanations
- [Andrej Karpathy — Neural Networks: Zero to Hero](https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ) — Build GPT from scratch
- [Yannic Kilcher](https://www.youtube.com/c/YannicKilcher) — ML paper reviews and explanations
- [Two Minute Papers](https://www.youtube.com/c/K%C3%A1rolyZsolnai) — AI research summaries

### Books
- [Deep Learning (Goodfellow)](https://www.deeplearningbook.org/) — Free online, the foundational textbook
- [Hands-On Machine Learning](https://www.oreilly.com/library/view/hands-on-machine-learning/9781098125967/) — Practical ML with scikit-learn and TensorFlow
- [Build a Large Language Model (From Scratch)](https://www.manning.com/books/build-a-large-language-model-from-scratch) — Understand LLMs by building one
