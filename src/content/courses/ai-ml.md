## Coming Soon

Build intelligent systems that learn from data. From classical machine learning algorithms through deep neural networks to modern LLM applications — understand how AI works and build with it.

This course covers ML fundamentals, PyTorch, transformer architectures, RAG pipelines, and deploying AI-powered applications in production.

## Top Resources to Get Started

### Official & Core Documentation
- [PyTorch Tutorials](https://pytorch.org/tutorials/) — Official deep learning tutorials from basics to advanced models
- [Hugging Face Documentation](https://huggingface.co/docs) — Transformers, datasets, model hub, and inference API
- [TensorFlow Documentation](https://www.tensorflow.org/learn) — Google's ML framework with tutorials and guides
- [OpenAI API Documentation](https://platform.openai.com/docs/) — GPT, DALL-E, and embeddings API reference
- [Anthropic API Documentation](https://docs.anthropic.com/) — Claude API, tool use, and prompt engineering guides
- [LangChain Documentation](https://python.langchain.com/docs/) — LLM application framework for chains, agents, and RAG
- [LlamaIndex Documentation](https://docs.llamaindex.ai/) — Data framework for building RAG and LLM applications

### Interactive Courses & Platforms
- [fast.ai](https://www.fast.ai/) — Practical deep learning, free and project-first with PyTorch and fastai
- [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) — Free ML fundamentals with TensorFlow
- [Hugging Face NLP Course](https://huggingface.co/learn/nlp-course) — Free NLP with Transformers, hands-on with real models
- [Kaggle Competitions](https://www.kaggle.com/competitions) — Real ML challenges with datasets, notebooks, and leaderboards
- [DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/) — Free bite-sized courses on LLMs, RAG, and prompt engineering
- [LLM Course (GitHub)](https://github.com/mlabonne/llm-course) — Free roadmap and Colab notebooks to get into Large Language Models
- [Learn PyTorch (Zero to Mastery)](https://www.learnpytorch.io/) — Hands-on PyTorch learning with code-first approach
- [Duke — Introduction to Machine Learning](https://www.coursera.org/learn/machine-learning-duke) — Free self-paced course covering CNNs, RNNs, and NLP with PyTorch

- [Andrej Karpathy — Neural Networks: Zero to Hero](https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ) — Build GPT from scratch, the best deep learning lecture series

- [AI Explained](https://www.youtube.com/@aiexplained-official) — Clear breakdowns of the latest AI developments and papers
- [freeCodeCamp — Machine Learning with Python](https://www.youtube.com/watch?v=i_LwzRVP7bg) — Full ML course covering neural networks, CNNs, and NLP

### Books & Long-Form Reading
- [Deep Learning (Goodfellow et al.)](https://www.deeplearningbook.org/) — Free online, the foundational deep learning textbook
- [Build a Large Language Model (From Scratch)](https://www.manning.com/books/build-a-large-language-model-from-scratch) — Understand LLMs by building a transformer with PyTorch
- [AI Engineering](https://www.oreilly.com/library/view/ai-engineering/9781098166298/) — Chip Huyen's guide to building AI systems in production
- [Hands-On Machine Learning (3rd Ed.)](https://www.oreilly.com/library/view/hands-on-machine-learning/9781098125967/) — Practical ML with scikit-learn, Keras, and TensorFlow
- [Natural Language Processing with Transformers](https://www.oreilly.com/library/view/natural-language-processing/9781098136789/) — By Hugging Face engineers, covers fine-tuning and inference
- [Speech and Language Processing (3rd Ed.)](https://web.stanford.edu/~jurafsky/slp3/) — Free draft by Jurafsky & Martin, covers transformers and LLMs
- [The LLM Engineer's Handbook](https://www.manning.com/books/the-llm-engineers-handbook) — Covers prompt engineering, fine-tuning, RAG, and deployment

- [Papers With Code](https://paperswithcode.com/) — ML papers with code implementations and benchmarks
