## Coming Soon

One codebase, every platform. Dart and Flutter let you build native mobile (iOS/Android), web, and desktop applications from a single project. Google's cross-platform framework with a strong "write once, run everywhere" story.

This course builds from Dart language fundamentals through Flutter's widget system to shipping cross-platform apps.

## Top Resources to Get Started

### Official & Core Documentation
- [Dart Language Tour](https://dart.dev/language) — Official overview of Dart's syntax, types, and core features
- [Dart API Reference](https://api.dart.dev/) — Complete API documentation for Dart's standard libraries
- [Flutter Documentation](https://docs.flutter.dev/) — Comprehensive guides, API reference, and cookbook for Flutter
- [Flutter Codelabs](https://docs.flutter.dev/codelabs) — Hands-on workshops from Google covering real app scenarios
- [Flutter Widget Catalog](https://docs.flutter.dev/ui/widgets) — Visual index of every Flutter widget with usage examples
- [pub.dev](https://pub.dev/) — Dart and Flutter's official package repository

### Interactive Courses & Platforms
- [DartPad](https://dartpad.dev/) — Write and run Dart and Flutter code directly in the browser
- [Exercism Dart Track](https://exercism.org/tracks/dart) — Mentored Dart exercises with community feedback
- [Flutter Learning Pathway](https://docs.flutter.dev/learn) — Google's recommended step-by-step path for learning Flutter
- [Google Codelabs — Flutter](https://codelabs.developers.google.com/?cat=Flutter) — Official hands-on tutorials covering layouts, animations, Firebase, and more
- [Flutter Apprentice (Kodeco)](https://www.kodeco.com/books/flutter-apprentice) — Project-based Flutter learning, building real apps from scratch
- [Dart Apprentice (Kodeco)](https://www.kodeco.com/books/dart-apprentice) — Dart language fundamentals through progressive exercises
- [Hello Dart (freeCodeCamp)](https://www.freecodecamp.org/) — Introduction to Dart through playful, interactive visualization
- [Educative — Learn Dart](https://www.educative.io/courses/learn-dart-first-step-to-flutter) — Free, interactive Dart course with in-browser coding
