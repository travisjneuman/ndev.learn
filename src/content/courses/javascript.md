## Coming Soon

The most versatile full-stack language. Frontend (React, Vue, Svelte), backend (Node.js, Express, NestJS), mobile (React Native), desktop (Electron), and serverless — all from one language.

This course starts with vanilla JavaScript fundamentals, progresses through TypeScript's type system, and builds full-stack applications across every platform.

## Top Resources to Get Started

### Official & Core Documentation
- [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) — The gold standard reference for JavaScript, maintained by Mozilla and the open-source community
- [MDN JavaScript Reference](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) — Complete API reference for every built-in object, method, and statement
- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/) — Official TypeScript documentation covering the entire type system
- [TypeScript Playground](https://www.typescriptlang.org/play) — Browser-based TypeScript editor with instant compilation and sharing
- [Node.js Documentation](https://nodejs.org/en/docs/) — Server-side JavaScript runtime docs with API reference and guides
- [ECMAScript Specification](https://tc39.es/ecma262/) — The living standard behind JavaScript, useful for understanding language internals

### Interactive Courses & Platforms
- [javascript.info](https://javascript.info/) — Modern JavaScript tutorial from beginner to advanced, widely considered the best free resource
- [freeCodeCamp JavaScript](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures-v8/) — Free, project-based JavaScript curriculum with certification
- [The Odin Project — Full Stack JavaScript](https://www.theodinproject.com/paths/full-stack-javascript) — Complete open-source curriculum from HTML to full-stack Node.js
- [TypeScript Exercises](https://typescript-exercises.github.io/) — Progressive TypeScript challenges that build on each other
- [Exercism JavaScript Track](https://exercism.org/tracks/javascript) — Mentored coding exercises with community feedback
- [Exercism TypeScript Track](https://exercism.org/tracks/typescript) — TypeScript-specific exercises with mentor reviews
- [Learn TypeScript](https://learntypescript.dev/) — Free TypeScript course for JavaScript developers with quizzes in each module
- [Scrimba — Learn JavaScript](https://scrimba.com/learn-javascript-c0v) — Interactive screencasts where you can edit code inside the video player

### Video Courses & YouTube Channels
- [Frontend Masters — JavaScript Learning Path](https://frontendmasters.com/learn/javascript/) — Expert-taught video courses from industry professionals
- [Fireship](https://www.youtube.com/@Fireship) — Fast-paced concept explainers and "100 Seconds" overviews for JS/TS and web tech
- [Traversy Media](https://www.youtube.com/@TraversyMedia) — Practical crash courses and project builds for JavaScript and frameworks
- [Jack Herrington](https://www.youtube.com/@jherr) — Advanced TypeScript patterns, React, and modern web architecture
- [Web Dev Simplified](https://www.youtube.com/@WebDevSimplified) — Clear explanations of JavaScript concepts from basics to advanced

- [Fun Fun Function](https://www.youtube.com/@funfunfunction) — Engaging explorations of functional programming and JavaScript concepts
- [Microsoft — TypeScript for Beginners](https://learn.microsoft.com/en-us/training/paths/build-javascript-applications-typescript/) — Free official Microsoft training modules for TypeScript

### Books & Long-Form Reading
- [Eloquent JavaScript](https://eloquentjavascript.net/) — Free online book, excellent for fundamentals and thinking programmatically
- [You Don't Know JS](https://github.com/getify/You-Dont-Know-JS) — Deep dive into JavaScript mechanics — closures, prototypes, async, and beyond
- [JavaScript Patterns](https://www.patterns.dev/) — Modern web design patterns for React and vanilla JS, free online
- [TypeScript Deep Dive](https://basarat.gitbook.io/typescript/) — Free, community-driven deep dive into TypeScript features and patterns
- [JavaScript: The Good Parts](https://www.oreilly.com/library/view/javascript-the-good/9780596517748/) — Douglas Crockford's classic on writing clean JavaScript
- [Programming TypeScript](https://www.oreilly.com/library/view/programming-typescript/9781492037644/) — O'Reilly guide to building robust TypeScript applications
