## Coming Soon

Memory safety without garbage collection. Rust gives you C-level performance with modern language ergonomics. Systems programming, CLIs, web services (Axum/Actix), and WebAssembly — all with the compiler catching bugs before they happen.

This course builds from ownership and borrowing through async Rust to production systems and WASM applications.

## Top Resources to Get Started

### Official & Core Documentation
- [The Rust Programming Language](https://doc.rust-lang.org/book/) — "The Book" — the definitive, community-maintained guide to learning Rust from scratch
- [Rust by Example](https://doc.rust-lang.org/rust-by-example/) — Learn Rust through annotated, runnable code examples
- [Rust Standard Library Documentation](https://doc.rust-lang.org/std/) — Complete API reference for the standard library
- [The Rustonomicon](https://doc.rust-lang.org/nomicon/) — Advanced guide to unsafe Rust and low-level details
- [Rust Reference](https://doc.rust-lang.org/reference/) — Formal specification of Rust syntax and semantics
- [The Cargo Book](https://doc.rust-lang.org/cargo/) — Official guide to Rust's package manager and build system

### Interactive Courses & Platforms
- [Rustlings](https://github.com/rust-lang/rustlings) — Official small exercises to learn Rust syntax and concepts, run from your terminal
- [100 Exercises to Learn Rust](https://rust-exercises.com/) — Luca Palmieri's progressive, hands-on Rust challenges
- [Exercism Rust Track](https://exercism.org/tracks/rust) — Mentored coding exercises with community feedback
- [Tour of Rust](https://tourofrust.com/) — Step-by-step interactive Rust tour in the browser
- [Comprehensive Rust (Google)](https://google.github.io/comprehensive-rust/) — Free course developed by Google's Android team, used internally at Google
- [Rust Playground](https://play.rust-lang.org/) — Write and run Rust code directly in the browser
- [CodeCrafters — Rust Track](https://codecrafters.io/tracks/rust) — Build your own Redis, Git, SQLite, and more from scratch in Rust
- [Advent of Code](https://adventofcode.com/) — Annual coding puzzles, popular in the Rust community for practice

### Video Courses & YouTube Channels
- [Let's Get Rusty](https://www.youtube.com/@letsgetrusty) — Follows "The Book" chapter by chapter with clear explanations
- [Jon Gjengset](https://www.youtube.com/@jonhoo) — Deep-dive intermediate and advanced Rust live coding and streams
- [No Boilerplate](https://www.youtube.com/@NoBoilerplate) — Compelling explanations of why Rust and how Rust concepts work
- [freeCodeCamp — Rust Course](https://www.youtube.com/watch?v=BpPEoZW5IiY) — Comprehensive free Rust course by Arfan Zubi
- [Traversy Media — Rust Crash Course](https://www.youtube.com/watch?v=zF34dRivLOw) — Quick introduction to Rust fundamentals
- [Tim McNamara (timClicks)](https://www.youtube.com/@timClicks) — Practical Rust tutorials and systems programming content
- [Code to the Moon](https://www.youtube.com/@codetothemoon) — Rust concepts, comparisons, and real-world applications
- [Ryan Levick](https://www.youtube.com/@RyanLevicksVideos) — Introduction to Rust series from a Microsoft engineer

### Books & Long-Form Reading
- [Programming Rust](https://www.oreilly.com/library/view/programming-rust-2nd/9781492052586/) — Comprehensive Rust from O'Reilly, covering the language in depth
- [Zero To Production In Rust](https://www.zero2prod.com/) — Build a real backend service in Rust from scratch, the best project-based Rust book
- [Rust in Action](https://www.manning.com/books/rust-in-action) — Systems programming projects covering networking, OS concepts, and more
- [Rust for Rustaceans](https://nostarch.com/rust-rustaceans) — Jon Gjengset's intermediate-level guide for those past the basics
- [Command-Line Rust](https://www.oreilly.com/library/view/command-line-rust/9781098109424/) — Learn Rust by building CLI tools that replace common Unix utilities
- [Hands-on Rust](https://pragprog.com/titles/hwrust/hands-on-rust/) — Learn Rust by building games with bracket-lib

### Community & Practice
- [Rust Users Forum](https://users.rust-lang.org/) — Official forum for questions, project ideas, and community discussion
- [Rust Discord](https://discord.gg/rust-lang) — Unofficial but highly active Discord with channels for help, projects, and Advent of Code
- [r/rust](https://www.reddit.com/r/rust/) — Active Reddit community for Rust news, projects, and discussion
- [r/learnrust](https://www.reddit.com/r/learnrust/) — Beginner-friendly subreddit for Rust learners
- [This Week in Rust](https://this-week-in-rust.org/) — Weekly newsletter with Rust news, articles, and community updates
- [crates.io](https://crates.io/) — Rust's package registry for discovering and sharing libraries
