## Coming Soon

Memory safety without garbage collection. Rust gives you C-level performance with modern language ergonomics. Systems programming, CLIs, web services (Axum/Actix), and WebAssembly — all with the compiler catching bugs before they happen.

This course builds from ownership and borrowing through async Rust to production systems and WASM applications.

## Top Resources to Get Started

### Official Documentation
- [The Rust Programming Language](https://doc.rust-lang.org/book/) — "The Book" — the definitive Rust guide
- [Rust by Example](https://doc.rust-lang.org/rust-by-example/) — Learn Rust through annotated examples
- [The Rustonomicon](https://doc.rust-lang.org/nomicon/) — Advanced unsafe Rust guide

### Interactive Courses
- [Rustlings](https://github.com/rust-lang/rustlings) — Small exercises to learn Rust syntax and concepts
- [Exercism Rust Track](https://exercism.org/tracks/rust) — Mentored Rust coding exercises
- [100 Exercises to Learn Rust](https://rust-exercises.com/) — Progressive Rust challenges
- [Tour of Rust](https://tourofrust.com/) — Step-by-step interactive Rust tour

### Video & Guided Learning
- [Let's Get Rusty](https://www.youtube.com/c/LetsGetRusty) — Follows "The Book" chapter by chapter
- [Jon Gjengset's Rust Streams](https://www.youtube.com/c/JonGjengset) — Deep-dive intermediate/advanced Rust
- [No Boilerplate](https://www.youtube.com/c/NoBoilerplate) — Why Rust, and Rust concepts explained

### Books
- [Programming Rust](https://www.oreilly.com/library/view/programming-rust-2nd/9781492052586/) — Comprehensive Rust from O'Reilly
- [Rust in Action](https://www.manning.com/books/rust-in-action) — Systems programming projects in Rust
- [Zero To Production In Rust](https://www.zero2prod.com/) — Build a real backend service in Rust
