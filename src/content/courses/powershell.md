## Coming Soon

The automation engine for Windows, Azure, and cross-platform systems administration. PowerShell turns complex manual processes into repeatable scripts — from managing Active Directory to deploying cloud infrastructure.

This course covers PowerShell from basics through advanced scripting, module development, and infrastructure automation.

## Top Resources to Get Started

### Official & Core Documentation
- [PowerShell Documentation](https://learn.microsoft.com/en-us/powershell/) — Microsoft's official PowerShell reference, guides, and release notes
- [PowerShell 101](https://learn.microsoft.com/en-us/powershell/scripting/learn/ps101/00-introduction) — Official beginner guide covering fundamentals step by step
- [PowerShell Cmdlet Reference](https://learn.microsoft.com/en-us/powershell/module/) — Complete reference for every PowerShell module and cmdlet
- [PowerShell Gallery](https://www.powershellgallery.com/) — Central repository for community modules, scripts, and DSC resources
- [PowerShell GitHub](https://github.com/PowerShell/PowerShell) — Open-source PowerShell 7+ source code, issues, and discussions
- [About Topics](https://learn.microsoft.com/en-us/powershell/module/microsoft.powershell.core/about/) — In-depth conceptual help on every PowerShell feature

### Books & Long-Form Reading
