## Coming Soon

Bridge the gap between code and production. DevOps practices turn manual deployments into automated pipelines. Docker containers, Kubernetes orchestration, CI/CD workflows, and infrastructure as code — the skills that keep modern software running.

This course builds from containerization basics through cloud deployment to production-grade infrastructure automation.

## Top Resources to Get Started

### Official & Core Documentation
- [Docker Documentation](https://docs.docker.com/) — Container fundamentals, Compose, and Swarm reference
- [Kubernetes Documentation](https://kubernetes.io/docs/) — Official K8s guides, tutorials, and API reference
- [Terraform Documentation](https://developer.hashicorp.com/terraform/docs) — Infrastructure as code with HCL
- [GitHub Actions Documentation](https://docs.github.com/en/actions) — CI/CD pipeline configuration and marketplace actions
- [Ansible Documentation](https://docs.ansible.com/) — Agentless automation and configuration management
- [Docker Roadmap](https://roadmap.sh/docker) — Visual guide to learning Docker step by step
- [DevOps Roadmap](https://roadmap.sh/devops) — Comprehensive visual guide to the full DevOps learning path

### Interactive Courses & Platforms
- [KodeKloud](https://kodekloud.com/) — Hands-on DevOps labs for Docker, K8s, Terraform, and Ansible (free courses available)
- [Docker Labs (Play with Docker)](https://labs.play-with-docker.com/) — Free Docker playground in the browser, no setup required
- [Kubernetes Playground](https://labs.play-with-k8s.com/) — Free K8s sandbox for experimenting with clusters
- [KubeAcademy](https://kube.academy/) — Free Kubernetes learning paths with virtual labs and exercises
- [DevOps with Kubernetes (University of Helsinki)](https://devopswithkubernetes.com/) — Free open-source course with certificate of completion
- [AWS Free Tier](https://aws.amazon.com/free/) — Hands-on cloud practice with 12 months of free services
- [freeCodeCamp DevOps](https://www.freecodecamp.org/news/tag/devops/) — Free DevOps articles, tutorials, and certification prep
- [Katacoda Scenarios](https://www.katacoda.com/) — Interactive browser-based labs for Docker, K8s, and CI/CD

### Video Courses & YouTube Channels
- [TechWorld with Nana](https://www.youtube.com/c/TechWorldwithNana) — DevOps tools explained clearly with full course playlists
- [NetworkChuck](https://www.youtube.com/c/NetworkChuck) — Docker, Linux, and cloud tutorials with high energy
- [CNCF YouTube Channel](https://www.youtube.com/c/cloudnativefdn) — KubeCon talks, Cloud Native Live sessions, and explainers
- [That DevOps Guy (Marcel Dempers)](https://www.youtube.com/c/MarcelDemworkers) — Kubernetes, Docker, and Terraform deep dives

- [freeCodeCamp — Docker Tutorial](https://www.youtube.com/watch?v=fqMOX6JJhGo) — Full Docker beginner course (over 2 hours)

### Books & Long-Form Reading
- [The Phoenix Project](https://itrevolution.com/product/the-phoenix-project/) — DevOps principles taught through a compelling novel
- [The Unicorn Project](https://itrevolution.com/product/the-unicorn-project/) — Developer-focused companion to The Phoenix Project
- [Docker Deep Dive](https://www.amazon.com/Docker-Deep-Dive-Nigel-Poulton/dp/1916585256) — Comprehensive Docker guide from basics to production
- [Kubernetes Up & Running (3rd Ed.)](https://www.oreilly.com/library/view/kubernetes-up-and/9781098110192/) — Production Kubernetes by the creators of K8s
- [Terraform: Up & Running (3rd Ed.)](https://www.terraformupandrunning.com/) — Hands-on IaC from "Hello, World" to full tech stack
- [Infrastructure as Code (2nd Ed.)](https://www.oreilly.com/library/view/infrastructure-as-code/9781098114664/) — Patterns and practices by Kief Morris
- [Site Reliability Engineering](https://sre.google/sre-book/table-of-contents/) — Free online, Google's definitive guide to running production systems
