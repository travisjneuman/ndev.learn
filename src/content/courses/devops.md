## Coming Soon

Bridge the gap between code and production. DevOps practices turn manual deployments into automated pipelines. Docker containers, Kubernetes orchestration, CI/CD workflows, and infrastructure as code — the skills that keep modern software running.

This course builds from containerization basics through cloud deployment to production-grade infrastructure automation.

## Top Resources to Get Started

### Official Documentation
- [Docker Documentation](https://docs.docker.com/) — Container fundamentals and reference
- [Kubernetes Documentation](https://kubernetes.io/docs/) — Official K8s guides and tutorials
- [Terraform Documentation](https://developer.hashicorp.com/terraform/docs) — Infrastructure as code
- [GitHub Actions Documentation](https://docs.github.com/en/actions) — CI/CD pipelines

### Interactive Courses
- [KodeKloud](https://kodekloud.com/) — Hands-on DevOps labs (Docker, K8s, Terraform)
- [Docker Labs](https://labs.play-with-docker.com/) — Free Docker playground in the browser
- [Kubernetes Playground](https://labs.play-with-k8s.com/) — Free K8s sandbox
- [AWS Free Tier](https://aws.amazon.com/free/) — Hands-on cloud practice
- [freeCodeCamp DevOps](https://www.freecodecamp.org/news/tag/devops/) — Free DevOps articles and tutorials

### Video & Guided Learning
- [TechWorld with Nana](https://www.youtube.com/c/TechWorldwithNana) — DevOps tools explained clearly
- [NetworkChuck](https://www.youtube.com/c/NetworkChuck) — Docker, Linux, and cloud tutorials
- [DevOps Roadmap](https://roadmap.sh/devops) — Visual guide to the DevOps learning path

### Books
- [The Phoenix Project](https://itrevolution.com/product/the-phoenix-project/) — DevOps principles through a novel
- [Docker Deep Dive](https://www.amazon.com/Docker-Deep-Dive-Nigel-Poulton/dp/1916585256) — Comprehensive Docker guide
- [Kubernetes Up & Running](https://www.oreilly.com/library/view/kubernetes-up-and/9781098110192/) — Production Kubernetes
