## Coming Soon

Turn raw data into actionable insights. Data science combines statistics, programming, and domain knowledge to extract meaning from datasets. Pandas for manipulation, Matplotlib and Seaborn for visualization, and scikit-learn for modeling.

This course covers the full data science workflow from data cleaning through exploratory analysis to building predictive models.

## Top Resources to Get Started

### Official & Core Documentation
- [Pandas Documentation](https://pandas.pydata.org/docs/) — Data manipulation library with getting started tutorials
- [NumPy Documentation](https://numpy.org/doc/) — Numerical computing fundamentals and array operations
- [scikit-learn User Guide](https://scikit-learn.org/stable/user_guide.html) — Machine learning algorithms with examples and theory
- [Matplotlib Documentation](https://matplotlib.org/stable/users/index.html) — Plotting and visualization reference
- [Seaborn Documentation](https://seaborn.pydata.org/) — Statistical data visualization built on Matplotlib
- [Jupyter Documentation](https://docs.jupyter.org/) — Interactive notebook environment for data exploration
- [Polars Documentation](https://docs.pola.rs/) — Fast DataFrame library as a modern Pandas alternative

### Interactive Courses & Platforms
- [Kaggle Learn](https://www.kaggle.com/learn) — Free micro-courses (Pandas, SQL, visualization, ML) with hands-on notebooks
- [freeCodeCamp — Data Analysis with Python](https://www.freecodecamp.org/learn/data-analysis-with-python/) — Free certification course with 5 real projects
- [Google Data Analytics Certificate](https://www.coursera.org/professional-certificates/google-data-analytics) — Professional certificate on Coursera (free to audit)
- [DataCamp](https://www.datacamp.com/) — Interactive data science courses with free introductory tier
- [Harvard CS109 — Data Science](https://cs109.github.io/2015/) — Full Harvard data science course materials, free online
- [MIT OpenCourseWare — Data Science](https://ocw.mit.edu/search/?q=data+science) — Free MIT course materials on statistics and data analysis
- [IBM Data Science Professional Certificate](https://www.coursera.org/professional-certificates/ibm-data-science) — Comprehensive program covering the full data science workflow

### Books & Long-Form Reading
- [Python for Data Analysis (3rd Ed.)](https://wesmckinney.com/book/) — Wes McKinney's Pandas bible, free to read online
- [Hands-On Machine Learning (3rd Ed.)](https://www.oreilly.com/library/view/hands-on-machine-learning/9781098125967/) — Scikit-learn, Keras, and TensorFlow by Aurelien Geron
- [Storytelling with Data](https://www.storytellingwithdata.com/) — Data visualization best practices and communication
- [Think Stats (2nd Ed.)](https://greenteapress.com/thinkstats2/html/) — Free online book on statistics for programmers using Python
- [Introduction to Statistical Learning](https://www.statlearning.com/) — Free textbook (ISLR) covering statistical learning methods with labs in R and Python
- [Naked Statistics](https://wwnorton.com/books/Naked-Statistics/) — Accessible introduction to statistics for non-math backgrounds
