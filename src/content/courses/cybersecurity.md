## Coming Soon

Understand how systems break so you can build them stronger. Cybersecurity covers network defense, web application security, cryptography, and ethical penetration testing — the skills that protect modern infrastructure.

This course builds from networking fundamentals through OWASP Top 10 to CTF challenges and security architecture design.

## Top Resources to Get Started

### Official & Core Documentation
- [OWASP Top 10](https://owasp.org/www-project-top-ten/) — The most critical web application security risks
- [NIST Cybersecurity Framework](https://www.nist.gov/cyberframework) — Security standards, guidelines, and best practices
- [MITRE ATT&CK](https://attack.mitre.org/) — Adversary tactics and techniques knowledge base used by defenders worldwide
- [CWE (Common Weakness Enumeration)](https://cwe.mitre.org/) — Catalog of software and hardware weakness types
- [OWASP Cheat Sheet Series](https://cheatsheetseries.owasp.org/) — Concise security guidance for specific topics (auth, crypto, input validation)
- [SANS Reading Room](https://www.sans.org/white-papers/) — Thousands of free cybersecurity research papers and whitepapers

### Interactive Courses & Platforms
- [TryHackMe](https://tryhackme.com/) — Guided cybersecurity labs with structured learning paths, beginner-friendly
- [Hack The Box](https://www.hackthebox.com/) — Hands-on hacking challenges, vulnerable machines, and pro labs
- [PortSwigger Web Security Academy](https://portswigger.net/web-security) — Free web security training with interactive labs (SQL injection, XSS, SSRF)
- [PicoCTF](https://picoctf.org/) — Beginner-friendly Capture The Flag challenges from Carnegie Mellon
- [OverTheWire Wargames](https://overthewire.org/wargames/) — Progressive security challenges via SSH (Bandit, Natas, Leviathan)
- [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) — Deliberately vulnerable web app for hands-on security testing
- [Fortinet Free Training](https://www.fortinet.com/training/cybersecurity-professionals) — Free cybersecurity courses and NSE certifications
- [Google Cybersecurity Certificate](https://www.coursera.org/professional-certificates/google-cybersecurity) — Beginner program covering security fundamentals and incident response

### Video Courses & YouTube Channels
- [Professor Messer — Security+](https://www.youtube.com/c/professormesser) — Free CompTIA Security+ training, comprehensive and well-structured
- [John Hammond](https://www.youtube.com/c/JohnHammond010) — CTF walkthroughs, malware analysis, and security research
- [The Cyber Mentor](https://www.youtube.com/c/TheCyberMentor) — Ethical hacking, penetration testing, and OSCP preparation
- [LiveOverflow](https://www.youtube.com/c/LiveOverflow) — Binary exploitation, reverse engineering, and security concepts
- [NetworkChuck](https://www.youtube.com/c/NetworkChuck) — Ethical hacking, networking, and Kali Linux tutorials
- [HackerSploit](https://www.youtube.com/c/HackerSploit) — Free infosec training covering penetration testing and red teaming
- [David Bombal](https://www.youtube.com/c/DavidBombal) — Networking, ethical hacking, and certification prep
- [MalwareTech](https://www.youtube.com/@MalwareTech) — Malware analysis, reverse engineering, and threat research

### Books & Long-Form Reading
- [The Web Application Hacker's Handbook (2nd Ed.)](https://www.wiley.com/en-us/The+Web+Application+Hacker%27s+Handbook-p-9781118026472) — The web security bible, essential for web app pentesting
- [Hacking: The Art of Exploitation (2nd Ed.)](https://nostarch.com/hacking2.htm) — Understanding exploits from the ground up with C and assembly
- [Penetration Testing](https://nostarch.com/pentesting) — Georgia Weidman's hands-on guide to pentesting methodology
- [Cybersecurity Ops with bash](https://www.oreilly.com/library/view/cybersecurity-ops-with/9781492041306/) — Security automation with the command line
- [The Art of Invisibility](https://www.littlebrown.com/titles/kevin-mitnick/the-art-of-invisibility/9780316380508/) — Kevin Mitnick's guide to privacy in the digital age
- [Network Security Essentials](https://www.pearson.com/en-us/subject-catalog/p/network-security-essentials/P200000003289) — William Stallings' guide to cryptography, firewalls, and IDS

### Community & Practice
- [r/netsec](https://www.reddit.com/r/netsec/) — Technical information security content and discussions
- [r/cybersecurity](https://www.reddit.com/r/cybersecurity/) — Broad cybersecurity community for news, career advice, and resources
- [CTFtime](https://ctftime.org/) — Global CTF competition calendar, team rankings, and writeups
- [HackerOne Hacktivity](https://hackerone.com/hacktivity) — Real-world vulnerability disclosures from bug bounty programs
- [The Hacker News](https://thehackernews.com/) — Cybersecurity news covering breaches, vulnerabilities, and malware
- [SANS Internet Storm Center](https://isc.sans.edu/) — Daily cybersecurity threat analysis and handler diaries
- [Security Stack Exchange](https://security.stackexchange.com/) — Expert Q&A for information security professionals
