## Coming Soon

Understand how systems break so you can build them stronger. Cybersecurity covers network defense, web application security, cryptography, and ethical penetration testing — the skills that protect modern infrastructure.

This course builds from networking fundamentals through OWASP Top 10 to CTF challenges and security architecture design.

## Top Resources to Get Started

### Official Documentation
- [OWASP Top 10](https://owasp.org/www-project-top-ten/) — The most critical web application security risks
- [NIST Cybersecurity Framework](https://www.nist.gov/cyberframework) — Security standards and guidelines
- [MITRE ATT&CK](https://attack.mitre.org/) — Adversary tactics and techniques knowledge base

### Interactive Courses
- [TryHackMe](https://tryhackme.com/) — Guided cybersecurity labs, beginner-friendly
- [Hack The Box](https://www.hackthebox.com/) — Hands-on hacking challenges and labs
- [OverTheWire Wargames](https://overthewire.org/wargames/) — Security challenges via SSH
- [PicoCTF](https://picoctf.org/) — Beginner-friendly Capture The Flag from Carnegie Mellon
- [PortSwigger Web Security Academy](https://portswigger.net/web-security) — Free web security training

### Video & Guided Learning
- [Professor Messer — Security+](https://www.youtube.com/c/professormesser) — Free CompTIA Security+ content
- [John Hammond](https://www.youtube.com/c/JohnHammond010) — CTF walkthroughs and security research
- [NetworkChuck](https://www.youtube.com/c/NetworkChuck) — Ethical hacking and networking
- [LiveOverflow](https://www.youtube.com/c/LiveOverflow) — Binary exploitation and security concepts

### Books
- [The Web Application Hacker's Handbook](https://www.wiley.com/en-us/The+Web+Application+Hacker%27s+Handbook-p-9781118026472) — Web security bible
- [Hacking: The Art of Exploitation](https://nostarch.com/hacking2.htm) — Understanding exploits from the ground up
- [Cybersecurity Ops with bash](https://www.oreilly.com/library/view/cybersecurity-ops-with/9781492041306/) — Security automation with the command line
