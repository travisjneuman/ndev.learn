## Coming Soon

The language of data. SQL powers every relational database from PostgreSQL to MySQL to SQLite. Learn to write queries, design schemas, optimize performance, and manage data at scale.

This course covers SQL fundamentals through advanced query optimization, database design patterns, and production PostgreSQL administration.

## Top Resources to Get Started

### Official & Core Documentation
- [PostgreSQL Documentation](https://www.postgresql.org/docs/) — Comprehensive PostgreSQL reference with tutorials, user guides, and internals
- [MySQL Reference Manual](https://dev.mysql.com/doc/refman/en/) — Official MySQL documentation covering all versions
- [SQLite Documentation](https://www.sqlite.org/docs.html) — Lightweight embedded database docs and SQL syntax reference
- [MariaDB Knowledge Base](https://mariadb.com/kb/en/) — Community-driven docs for the MySQL fork
- [W3Schools SQL Tutorial](https://www.w3schools.com/sql/) — Quick reference with a "Try it Yourself" editor for every concept
- [SQL Style Guide](https://www.sqlstyle.guide/) — Consistent formatting conventions for readable SQL

### Interactive Courses & Platforms
- [SQLBolt](https://sqlbolt.com/) — Interactive SQL lessons with live exercises, great for absolute beginners
- [Select Star SQL](https://selectstarsql.com/) — Interactive SQL book using real Texas death row data
- [SQLZoo](https://sqlzoo.net/) — Interactive tutorials and quizzes covering joins, subqueries, and aggregates
- [PostgreSQL Exercises](https://pgexercises.com/) — Practice PostgreSQL queries on a realistic club management dataset
- [SQL Easy](https://www.sql-easy.com/) — Clean, beginner-friendly interactive SQL tutorial
- [HackerRank SQL](https://www.hackerrank.com/domains/sql) — SQL challenges from beginner to advanced with instant feedback
- [LeetCode Database Problems](https://leetcode.com/problemset/database/) — SQL interview-style problems used by top tech companies
- [Oracle Dev Gym — Databases for Developers](https://devgym.oracle.com/pls/apex/dg/class/databases-for-developers-foundations.html) — Free 12-part Oracle SQL course with certificate

### Video Courses & YouTube Channels
- [CMU Database Course (Andy Pavlo)](https://15445.courses.cs.cmu.edu/) — Carnegie Mellon's database systems course, free lectures covering internals
- [Hussein Nasser — Database Engineering](https://www.youtube.com/c/HusseinNasser-software-engineering) — Deep dives into database internals, indexing, and ACID properties
- [freeCodeCamp — SQL Full Course](https://www.youtube.com/watch?v=HXV3zeQKqGY) — Complete 4-hour SQL beginner course
- [Fireship — SQL in 100 Seconds](https://www.youtube.com/watch?v=zsjvFFKOm3c) — Quick conceptual overview of SQL
- [Corey Schafer — SQLite & Python](https://www.youtube.com/watch?v=pd-0G0MigUA) — Practical database usage in real applications
- [Khan Academy — Intro to SQL](https://www.khanacademy.org/computing/computer-programming/sql) — Video tutorials paired with interactive coding challenges

### Books & Long-Form Reading
- [Learning SQL (3rd Ed.)](https://www.oreilly.com/library/view/learning-sql-3rd/9781492057604/) — O'Reilly's practical guide from basics to advanced queries
- [SQL Performance Explained](https://sql-performance-explained.com/) — Deep dive into indexing and query optimization by Markus Winand
- [Designing Data-Intensive Applications](https://dataintensive.net/) — Martin Kleppmann's essential guide to data systems architecture
- [Practical SQL (2nd Ed.)](https://nostarch.com/practical-sql-2nd-edition) — Learn SQL through real-world datasets (census, earthquake, taxi data)
- [SQL Antipatterns](https://pragprog.com/titles/bksqla/sql-antipatterns/) — Common SQL mistakes and how to avoid them
- [SQL Cookbook (2nd Ed.)](https://www.oreilly.com/library/view/sql-cookbook-2nd/9781492077435/) — Favorite SQL techniques including window functions and pivoting

### Community & Practice
- [Kaggle SQL Micro-Course](https://www.kaggle.com/learn/intro-to-sql) — Free hands-on SQL with BigQuery on real datasets
- [Exercism SQL Track](https://exercism.org/tracks/plsql) — Mentored SQL exercises with community code review
- [r/SQL](https://www.reddit.com/r/SQL/) — Active community for SQL questions, tips, and career advice
- [Database Administrators Stack Exchange](https://dba.stackexchange.com/) — Expert Q&A for database professionals
- [Use The Index, Luke](https://use-the-index-luke.com/) — Free web book on SQL indexing and tuning by Markus Winand
- [DataCamp SQL Courses](https://www.datacamp.com/courses/introduction-to-sql) — Interactive browser-based SQL with free introductory tier
