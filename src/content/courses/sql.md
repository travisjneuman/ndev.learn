## Coming Soon

The language of data. SQL powers every relational database from PostgreSQL to MySQL to SQLite. Learn to write queries, design schemas, optimize performance, and manage data at scale.

This course covers SQL fundamentals through advanced query optimization, database design patterns, and production PostgreSQL administration.

## Top Resources to Get Started

### Official Documentation
- [PostgreSQL Documentation](https://www.postgresql.org/docs/) — Comprehensive PostgreSQL reference
- [SQLite Documentation](https://www.sqlite.org/docs.html) — Lightweight database docs
- [MySQL Reference Manual](https://dev.mysql.com/doc/refman/en/) — Official MySQL documentation

### Interactive Courses
- [SQLBolt](https://sqlbolt.com/) — Interactive SQL lessons with live exercises
- [Select Star SQL](https://selectstarsql.com/) — Interactive SQL book using real data
- [SQLZoo](https://sqlzoo.net/) — Interactive SQL tutorials and quizzes
- [PostgreSQL Exercises](https://pgexercises.com/) — Practice PostgreSQL queries on a real dataset
- [Exercism SQL Track](https://exercism.org/tracks/plsql) — Mentored SQL exercises

### Video & Guided Learning
- [CMU Database Course](https://15445.courses.cs.cmu.edu/) — Andy Pavlo's database systems course (free lectures)
- [Hussein Nasser — Database Engineering](https://www.youtube.com/c/HusseinNasser-software-engineering) — Deep database internals
- [freeCodeCamp — SQL Tutorial](https://www.youtube.com/watch?v=HXV3zeQKqGY) — Full SQL beginner course

### Books
- [Learning SQL](https://www.oreilly.com/library/view/learning-sql-3rd/9781492057604/) — O'Reilly's practical SQL guide
- [Designing Data-Intensive Applications](https://dataintensive.net/) — Martin Kleppmann's essential data systems book
- [SQL Performance Explained](https://sql-performance-explained.com/) — Index and query optimization
