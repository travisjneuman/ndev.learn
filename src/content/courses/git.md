## Coming Soon

Every developer's most essential tool. Git tracks changes, enables collaboration, and powers the entire open-source ecosystem. Learn branching strategies, merge conflict resolution, and professional GitHub workflows.

This course covers Git from first commit through advanced rebasing, CI/CD integration, and team collaboration patterns.

## Top Resources to Get Started

### Official & Core Documentation
- [Pro Git Book](https://git-scm.com/book/en/v2) — The free, official Git book by Scott Chacon and Ben Straub — comprehensive and well-written
- [Git Reference](https://git-scm.com/docs) — Complete command reference with examples for every Git command
- [GitHub Docs](https://docs.github.com/) — Pull requests, Actions, collaboration features, and API reference
- [GitLab Docs](https://docs.gitlab.com/) — Alternative platform docs covering Git workflows and CI/CD
- [Conventional Commits](https://www.conventionalcommits.org/) — Commit message standard used across the industry
- [Git Cheat Sheet (GitHub)](https://education.github.com/git-cheat-sheet-education.pdf) — Quick printable command reference

### Interactive Courses & Platforms
- [Learn Git Branching](https://learngitbranching.js.org/) — Visual, interactive tutorial for branching, merging, rebasing, and cherry-picking
- [GitHub Skills](https://skills.github.com/) — Official GitHub learning paths with automated feedback inside real repos
- [Oh My Git!](https://ohmygit.org/) — Open-source game that visualizes repository internals in real time
- [Git Exercises](https://gitexercises.fracz.com/) — Practice real Git scenarios hands-on with progressive difficulty
- [Codecademy — Learn Git](https://www.codecademy.com/learn/learn-git) — Interactive browser-based Git fundamentals course
- [W3Schools Git Tutorial](https://www.w3schools.com/git/) — Step-by-step Git tutorial with "Try it Yourself" exercises
- [Katacoda Git Scenarios](https://www.katacoda.com/courses/git) — Browser-based terminal labs for Git practice
- [Git Tower — Learn Git](https://www.git-tower.com/learn/) — Free video course with 24 episodes averaging 5 minutes each

### Video Courses & YouTube Channels
- [The Missing Semester — Git (MIT)](https://missing.csail.mit.edu/2020/version-control/) — MIT's Git lecture covering data model and internals
- [freeCodeCamp — Git and GitHub for Beginners](https://www.youtube.com/watch?v=RGOj5yH7evk) — Complete beginner tutorial (over 1 hour)
- [Fireship — Git in 100 Seconds](https://www.youtube.com/watch?v=hwP7WQkmECE) — Quick conceptual overview
- [Net Ninja — Git & GitHub Tutorial](https://www.youtube.com/playlist?list=PL4cUxeGkcC9goXbgTDQ0n_4TBzOO0ocPR) — Beginner-friendly playlist covering essential workflows
- [ThePrimeagen — Everything You'll Need to Know About Git](https://www.youtube.com/watch?v=ecK3EnyGD8o) — Advanced Git concepts from a Netflix engineer
- [Google — Introduction to Git and GitHub (Coursera)](https://www.coursera.org/learn/introduction-git-github) — Free to audit, covers branching and merging workflows

### Books & Long-Form Reading
- [Pro Git (2nd Ed.)](https://git-scm.com/book/en/v2) — Free and definitive, covers everything from basics to Git internals
- [Git for Teams](https://gitforteams.com/) — Collaboration strategies, branching models, and team workflows
- [Version Control with Git (3rd Ed.)](https://www.oreilly.com/library/view/version-control-with/9781492091189/) — O'Reilly guide covering advanced Git operations
- [Git Pocket Guide](https://www.oreilly.com/library/view/git-pocket-guide/9781449327507/) — Concise quick-reference for everyday Git commands
- [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials) — Well-written guides on Git workflows, branching strategies, and advanced topics

### Community & Practice
- [r/git](https://www.reddit.com/r/git/) — Active community for Git questions, tips, and workflow discussions
- [GitHub Community Discussions](https://github.com/orgs/community/discussions) — Official GitHub community forum for questions and feature discussions
- [Stack Overflow — Git Tag](https://stackoverflow.com/questions/tagged/git) — Thousands of answered Git questions with expert solutions
- [Git Flight Rules](https://github.com/k88hudson/git-flight-rules) — Community-maintained guide for "what to do when things go wrong in Git"
- [Oh Shit, Git!?!](https://ohshitgit.com/) — Plain-English solutions for common Git mistakes
- [Gitconnected](https://gitconnected.com/learn/git) — Curated collection of the best Git tutorials and courses
