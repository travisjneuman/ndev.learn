## Coming Soon

Google's preferred language for Android development, but Kotlin goes far beyond mobile. Backend (Ktor, Spring), multiplatform (KMP), Compose Multiplatform for shared UI, and full JVM interop.

This course builds from Kotlin fundamentals through Android development to multiplatform applications.

## Top Resources to Get Started

### Official Documentation
- [Kotlin Documentation](https://kotlinlang.org/docs/home.html) — Official Kotlin docs and tutorials
- [Kotlin Koans](https://kotlinlang.org/docs/koans.html) — Interactive Kotlin exercises in the browser
- [Android Kotlin Fundamentals](https://developer.android.com/courses/kotlin-fundamentals/course) — Google's official Android + Kotlin course

### Interactive Courses
- [Exercism Kotlin Track](https://exercism.org/tracks/kotlin) — Mentored Kotlin exercises
- [JetBrains Academy — Kotlin](https://hyperskill.org/tracks/3) — Project-based Kotlin learning
- [Kotlin Playground](https://play.kotlinlang.org/) — Try Kotlin in the browser
- [Google Android Basics with Compose](https://developer.android.com/courses/android-basics-compose/course) — Official Compose course
