## Coming Soon

Google's preferred language for Android development, but Kotlin goes far beyond mobile. Backend (Ktor, Spring), multiplatform (KMP), Compose Multiplatform for shared UI, and full JVM interop.

This course builds from Kotlin fundamentals through Android development to multiplatform applications.

## Top Resources to Get Started

### Official & Core Documentation
- [Kotlin Documentation](https://kotlinlang.org/docs/home.html) — Official Kotlin language reference, tutorials, and guides
- [Kotlin API Reference](https://kotlinlang.org/api/latest/jvm/stdlib/) — Complete standard library API documentation
- [Kotlin Koans](https://kotlinlang.org/docs/koans.html) — 40+ interactive exercises in the browser for learning Kotlin syntax and idioms
- [Android Kotlin Fundamentals](https://developer.android.com/courses/kotlin-fundamentals/course) — Google's official Android + Kotlin course
- [Kotlin Multiplatform Documentation](https://kotlinlang.org/docs/multiplatform.html) — Official docs for sharing code across platforms
- [Ktor Documentation](https://ktor.io/docs/welcome.html) — JetBrains' official guide to building server-side applications with Kotlin

### Interactive Courses & Platforms
- [Google Android Basics with Compose](https://developer.android.com/courses/android-basics-compose/course) — Official free course for building Android apps with Jetpack Compose
- [JetBrains Academy — Kotlin](https://hyperskill.org/tracks/3) — Project-based Kotlin learning with real-world applications
- [Exercism Kotlin Track](https://exercism.org/tracks/kotlin) — Mentored coding exercises with community feedback
- [Kotlin Playground](https://play.kotlinlang.org/) — Write and run Kotlin code directly in the browser
- [Google Udacity — Developing Android Apps with Kotlin](https://www.udacity.com/course/developing-android-apps-with-kotlin--ud9012) — Free course from Google on Android development
- [Kotlin by Example](https://play.kotlinlang.org/byExample/overview) — Interactive examples covering Kotlin features
- [Codewars Kotlin](https://www.codewars.com/?language=kotlin) — Progressive coding challenges in Kotlin
- [Educative — Learn Kotlin](https://www.educative.io/courses/learn-kotlin-from-scratch) — Interactive Kotlin course with zero setup required
