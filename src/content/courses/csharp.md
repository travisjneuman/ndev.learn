## Coming Soon

Backend (ASP.NET), desktop (WPF/MAUI), game development (Unity), and web frontend (Blazor) — C# and .NET cover an enormous surface area from a single language ecosystem.

This course builds from language fundamentals through enterprise patterns to shipping games and cross-platform apps.

## Top Resources to Get Started

### Official Documentation
- [C# Documentation](https://learn.microsoft.com/en-us/dotnet/csharp/) — Microsoft's official C# docs
- [.NET Learning Center](https://dotnet.microsoft.com/en-us/learn) — Guided paths from Microsoft
- [Unity Learn](https://learn.unity.com/) — Official Unity tutorials and courses

### Interactive Courses
- [Exercism C# Track](https://exercism.org/tracks/csharp) — Mentored C# coding exercises

- [C# Fundamentals on Pluralsight](https://www.pluralsight.com/paths/c-sharp) — Structured learning path
