## Coming Soon

Backend (ASP.NET), desktop (WPF/MAUI), game development (Unity), and web frontend (Blazor) — C# and .NET cover an enormous surface area from a single language ecosystem.

This course builds from language fundamentals through enterprise patterns to shipping games and cross-platform apps.

## Top Resources to Get Started

### Official & Core Documentation
- [C# Documentation](https://learn.microsoft.com/en-us/dotnet/csharp/) — Microsoft's official C# language reference and tutorials
- [.NET Learning Center](https://dotnet.microsoft.com/en-us/learn) — Guided learning paths from Microsoft covering C#, ASP.NET, Blazor, and MAUI
- [C# Language Tour](https://learn.microsoft.com/en-us/dotnet/csharp/tour-of-csharp/) — Quick overview of C# syntax, types, and features
- [Unity Learn](https://learn.unity.com/) — Official Unity tutorials and courses for game development with C#
- [ASP.NET Documentation](https://learn.microsoft.com/en-us/aspnet/core/) — Official docs for building web apps, APIs, and microservices with C#
- [.NET API Browser](https://learn.microsoft.com/en-us/dotnet/api/) — Complete API reference for the entire .NET class library

### Interactive Courses & Platforms
- [Foundational C# with Microsoft (freeCodeCamp)](https://www.freecodecamp.org/learn/foundational-c-sharp-with-microsoft/) — Free, official Microsoft-backed C# curriculum with certification
- [Microsoft Learn — C# Path](https://learn.microsoft.com/en-us/training/paths/get-started-c-sharp-part-1/) — Free interactive modules from beginner to advanced
- [Exercism C# Track](https://exercism.org/tracks/csharp) — Mentored coding exercises with community feedback
- [Codecademy Learn C#](https://www.codecademy.com/learn/learn-c-sharp) — Interactive beginner course covering fundamentals
- [Learn C# (learncs.org)](https://www.learncs.org/) — Free interactive tutorial for learning C# in the browser
- [Codewars C#](https://www.codewars.com/?language=csharp) — Progressive coding challenges from beginner to expert
- [Dometrain — C# for Beginners](https://dometrain.com/course/hands-on-csharp-for-beginners/) — Free hands-on course from Nick Chapsas
- [C# Fundamentals on Pluralsight](https://www.pluralsight.com/paths/c-sharp) — Structured learning path with video instruction

### Books & Long-Form Reading
- [C# in Depth](https://csharpindepth.com/) — Jon Skeet's authoritative deep dive into C# language features and evolution
- [Head First C#](https://www.oreilly.com/library/view/head-first-c/9781098141783/) — Visual, hands-on approach to learning C# through projects
- [C# 12 and .NET 8](https://www.packtpub.com/product/c-12-and-net-8-modern-cross-platform-development-fundamentals/9781837635870) — Mark Price's comprehensive modern C# guide updated annually
- [Concurrency in C# Cookbook](https://www.oreilly.com/library/view/concurrency-in-c/9781492054498/) — Stephen Cleary's guide to async, parallel, and reactive patterns
- [CLR via C#](https://www.microsoftpressstore.com/store/clr-via-c-sharp-9780735667457) — Jeffrey Richter's deep dive into the Common Language Runtime
- [Home and Learn C#](https://www.homeandlearn.co.uk/csharp/csharp.html) — Free online course and tutorial for complete beginners
